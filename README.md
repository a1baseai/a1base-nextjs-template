# A1Base AI Chat Agent Template

This template demonstrates how to build an AI-powered chat agent using [A1Base](https://a1base.com) and [Next.js](https://nextjs.org). The agent can communicate through multiple channels including WhatsApp, Email, Slack, Teams, and SMS, with WhatsApp integration showcased in this demo.
